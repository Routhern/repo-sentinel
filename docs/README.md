# 문서

repo-sentinel의 상세 문서 모음이다. 빠른 시작은 저장소 루트의
[`README.md`](../README.md)를 참고하라.

- [`commands.md`](commands.md) — 전체 CLI 명령어 참고 (옵션, 단축 별칭,
  TUI 안에서의 사용법 포함).

  데이터 흐름, NAS와의 책임 분리, TUI 구조.

프로젝트 기획 배경 전문은 [`CONTEXTS_AND_PLAN.md`](../CONTEXTS_AND_PLAN.md),
버전별 변경 이력은 [`CHANGELOG.md`](../CHANGELOG.md), 개발 환경 설정과
기여 방법은 [`CONTRIBUTING.md`](../CONTRIBUTING.md)에 있다.
